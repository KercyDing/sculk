# sculk-tui

Minecraft P2P multiplayer tunnel TUI client, built on [sculk](https://crates.io/crates/sculk) + [ratatui](https://crates.io/crates/ratatui).

## Installation

```sh
cargo install sculk-tui
```

## Usage

```sh
sculk-tui
```

Three modes: `Host / Join / Relay`

### Key Bindings

| Key | Action |
|---|---|
| `←` / `→` | Switch mode |
| `Tab` | Toggle focus (config / log) |
| `↑` / `↓` | Navigate fields or scroll log |
| `Enter` | Start/stop tunnel, or apply relay |
| `e` | Edit field |
| `q` | Quit editing |
| `h` | Help |
| `c` | Clear log |
| `Esc×2` | Exit |
